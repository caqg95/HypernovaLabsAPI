# Hypernova Labs API Gestion de Gastos

Este proyecto nos ayuda a gestionar los Gastos de la Empresa. Se tomo en cuenta tecnologia  como JWT, Identity , Sql Server ,SwaggerDoc.

## Intrucciones

Para evitar problemas en la ejecución, necesitamos contar con:

1-Visual Studio 2019 o Visual Studio Code

2-Sql Server

3-.Net Core 3.1 SDK

4-Postman

5-.NET Core Hosting Bundle(IIS)

## Nota

*  Debemos de  modificar las credenciales en la Cadena de Conexión en **appsettings.json** con la del nuevo servidor de BD.
* En el Proyecto tiene el Script de la DB , El archivo lo encontramos en **/DB/DBHYPERNOVALABS.sql** 
* En el servidor de DB , debemos crear la base de dato con el siguiente nombre **HYPERNOVALABS** ,siguiente a esto ejecutamos el Script
